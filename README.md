## Description

Tic-Tac-Toe Game is a simple and fun game developed using HTML, CSS, and JavaScript. It is a two-player game where players take turns marking spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.
